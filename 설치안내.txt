============================================================
  MBAM 마케팅 프로그램 - 다른 컴퓨터(노트북) 설치 안내
============================================================

[ 준비물 (노트북에 미리 설치) ]
  1) Python 3.12 이상   → https://www.python.org/downloads/
     - 설치 화면에서 "Add Python to PATH" 반드시 체크!
  2) Node.js LTS        → https://nodejs.org
  3) (선택) USB 테더링 IP 우회 기능을 쓰려면: ADB(플랫폼 툴) + 안드로이드폰 USB 디버깅
     → https://developer.android.com/tools/releases/platform-tools


[ 옮길 파일 ]
  이 프로그램 폴더(마케팅 프로그램) 전체를 노트북으로 복사하세요. (USB / 외장하드 / 클라우드)
  단, 아래 두 폴더는 용량이 크고 새로 설치되므로 빼고 복사하면 빠릅니다.
     - venv\              (설치하기.bat 가 새로 만듭니다)
     - mbam-web\node_modules\   (설치하기.bat 가 새로 받습니다)


[ 설치 방법 ]
  1) 복사한 폴더에서  "설치하기.bat"  을 더블클릭
     → 가상환경 생성 + 파이썬/화면 패키지 설치 + 브라우저 설치 + 보안키 생성 (한 번만)
  2) 끝나면  "시작하기.bat"  더블클릭
     → 백엔드/화면 서버가 켜지고 브라우저(http://localhost:3000)가 자동으로 열림


[ 현재 설정 그대로 쓰려면 (중요) ]
  - .env 파일에 네이버 API 키 등 "현재 설정"이 들어 있습니다. 폴더와 함께 복사되면 그대로 적용됩니다.
  - 기존 계정/이력 DB(seo_history.db, mbam_local.db)도 함께 복사하면 데이터가 유지됩니다.
    단, 계정 비밀번호 복호화를 위해 원본 PC의 보안키가 같아야 합니다:
       · 원본 PC의 JWT_SECRET / ENV_CIPHER_KEY 값을 .env 에 그대로 적어 넣으세요.
       · 값을 모르거나 새로 시작해도 되면, 설치하기.bat 가 새 키를 자동 생성합니다(계정은 새로 등록).


[ 네이버 계정 (꼭 확인) ]
  네이버 "기기 인증"은 컴퓨터마다 따로입니다. 노트북에서는
     [설정 → 계정관리 → 기기인증] 을 계정마다 1회 다시 진행해야 자동 로그인이 됩니다.
  (블로그/카페 발행·댓글 등 로그인 자동화 기능은 기기인증이 된 계정에서만 동작)


[ 자주 묻는 문제 ]
  - "python/node 가 없다" 라고 뜨면 → 준비물 1·2를 설치하고 PATH 체크 후 다시 실행
  - 화면이 안 열리면 → 잠시 기다린 뒤(첫 실행은 준비 10~20초) http://localhost:3000 직접 접속
  - 포트 충돌 → 시작하기.bat 가 8000/3000 기존 서버를 자동 정리합니다
  - 방화벽 경고가 뜨면 "허용"을 눌러주세요 (로컬 서버 통신용)
