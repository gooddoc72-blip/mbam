# [프로그램명] 제품 소개 핵심 자료

## 1. 한 줄 정의
[프로그램명]은 글감 수집부터 AI 원고 작성과 발행까지 매일 자동으로 처리하는 데이터 기반 올인원 마케팅 자동화 프로그램이다. 네이버 블로그, 카페, 플레이스, 쇼핑을 한 화면에서 운영하여 1인이 마케팅팀처럼 일할 수 있게 한다.

## 2. 해결하는 문제
- 소상공인은 블로그가 매출에 중요하다는 것을 알지만 글을 쓸 시간이 없고 무엇을 써야 할지 막막하다.
- 마케팅 대행은 비용이 높고(월 수십만 원대) 효과를 확인하기 어렵다.
- 마케터와 대행사는 다계정에 매일 수동으로 발행하느라 인력이 부족하다.
- 글을 써도 상위노출이 되지 않는 이유를 알기 어렵다.

## 3. 핵심 특징

### 3.1 매일 자동발행 (블로그 / 카페)
- 글감 카테고리와 발행 시각을 설정하면 매일 같은 시각에 글감 선택, AI 원고 작성, 발행이 자동으로 이루어진다.
- 매일 다른 글감으로 회전하며 하루 1회 중복 발행을 방지한다.
- 첨부 이미지가 없으면 글 제목과 소제목 기반으로 카드뉴스 이미지 5장을 자동 생성해 본문에 삽입한다.
- 다계정 순차 발행을 지원한다.
- 카페는 글 발행뿐 아니라 방문, 좋아요 부스트, 댓글 육성까지 예약 자동화한다.

### 3.2 데이터 기반 글쓰기 (상위노출 공식 복제)
- 카페글과 블로그글 분석: 인기 글의 URL을 입력하면 글 유형(카페/블로그)을 자동 판별하여 각각에 맞는 5가지 지표로 해부한다.
  - 지표: 검색의도 정합성(C-Rank), 본문 구조와 가독성, 경험 데이터(D.I.A+), 이웃/댓글 유도, 작성자 영향력.
- 분석 결과에서 "이 분석으로 글쓰기" 기능으로 곧바로 동일 구조의 새 글을 작성한다.
- SEO 분석: 검색 상위 글의 키워드, 구조, 작성자 통계를 벤치마킹한다.

### 3.3 글감 자동 수집과 황금키워드
- 소상공인24, 기업마당, 창업진흥원, 복지로, 공연/행사, 뉴스, 캠핑, 생태관광, 반려동물 동반여행, 웰니스, 국민임대, 행복주택 등 카테고리별 글감을 매일 수집한다.
- 황금키워드(검색량 대비 경쟁이 낮은 키워드)를 추천한다.
- 관심 카테고리 요약을 텔레그램으로 알림 전송한다.

### 3.4 AI 원고 엔진 (선택형)
- Claude, Gemini, ChatGPT 중 선택해 사용하며, 사용자 본인의 API 키를 쓰는 방식(BYOK)으로 비용이 투명하다.
- 포스팅 목적(리뷰, 홍보, 정보성)과 유형(맛집, 병원, 상품, 앱/서비스, 서비스업), 배포 방식(상세형, 막배포)을 선택할 수 있다.
- 상품 URL을 입력하면 상품 정보와 이미지를 자동 스크랩해 글 작성에 활용한다.

### 3.5 순위와 진단 분석
- 플레이스 진단: 네이버 지도 순위를 추적하고 매일 새벽 자동 갱신하며 히스토리 그래프를 제공한다.
- 쇼핑 순위 검색, 쇼핑 키워드 분석, 상품명 키워드 조합, 쿠팡 순위 분석을 제공한다.
- 블로그 지수 진단, 구글 블로그스팟 자동화를 제공한다.

### 3.6 부가 도구
- 이미지 세탁소: 중복 회피를 위한 이미지 변형.
- 카드뉴스 자동 생성.
- 플레이스 소식과 숏폼 영상 자동 제작: 리뷰를 분석해 새소식 원고와 홍보 클립을 만든다.
- 소통 및 이웃 자동화.

### 3.7 운영 편의 기능
- 다계정 기기 인증: 1회 로그인으로 2단계 인증을 회피하고 자동 로그인 상태를 유지한다.
- USB 테더링 IP 로테이션으로 계정별 IP를 분리한다.
- 예약 발행(원하는 날짜와 시간 지정)과 내장 스케줄러(서버를 켜두면 매일 자동 실행).
- 메뉴별 작업내역 리스트와 다시 보기, 메뉴 이동 시 작업 상태 유지, 대시보드 통합 현황.

## 4. 활용 시나리오
- 시간이 없는 소상공인: 글감 카테고리와 발행 시각만 설정하면 매일 자동으로 블로그가 채워진다.
- 신규 매장 오픈: 상품 또는 매장 URL을 입력해 정보와 이미지를 자동 수집하고 홍보/맛집 유형으로 글을 대량 생성한다.
- 순위가 오르지 않을 때: 1등 글 URL을 분석해 성공 공식을 추출하고 그대로 새 글을 작성한다.
- 대행사를 쓰던 사장님: 대행비 없이 직접 운영하며 순위를 수치로 확인한다.
- 마케터와 대행사: 계정을 기기 인증한 뒤 다계정 예약과 매일 자동발행, 카페 육성을 일괄 처리한다.
- 플레이스 운영자: 리뷰 기반 새소식 원고와 숏폼 클립을 자동 제작한다.

## 5. 도입 효과
- 시간: 글 1건 기획부터 작성, 발행까지 걸리던 시간을 사실상 0에 가깝게(무인) 줄인다. 〈평균 45분 → 약 0분〉
- 비용: 블로그 대행 비용(〈월 80만 원대〉)이나 콘텐츠 외주 비용을 자체 운영으로 대체하며, AI 비용만 본인 키로 투명하게 지출한다.
- 성과: 데이터 기반 작성으로 노출 확률을 높이고 플레이스/쇼핑 순위를 수치로 추적한다.
- 규모: 1인이 블로그, 카페, 플레이스, 쇼핑, 블로그스팟을 동시에 운영해 마케팅팀 수준의 산출량을 낸다.

## 6. 편리함 요약
- 켜두면 글감 수집부터 발행까지 무인 자동 처리.
- 분석 결과에서 글쓰기로 원클릭 연결.
- 분석, 작성, 발행, 순위를 한 화면에서 통합 관리.
- 작업내역 자동 저장과 다시 보기로 과거 작업 복원.
- 계정 1회 인증 후 자동 로그인.
- 카드뉴스 생성과 이미지 세탁 내장.
- 텔레그램 알림 연동.

## 7. 차별점
- 초안만 보조하는 일반 도구와 달리 수집부터 작성, 발행까지 완전 자동.
- 감이 아니라 상위노출 글을 해부해 공식을 복제하는 데이터 기반 작성.
- 채널별로 분리된 도구가 아니라 블로그, 카페, 플레이스, 쇼핑 통합 운영.
- 단발 작업이 아니라 스케줄러로 매일 무인 운영.
- 글뿐 아니라 이미지와 영상, 순위 추적까지 포함.

## 8. 타깃 고객
- 소상공인 및 자영업자: 블로그 운영 시간이 없는 사업주.
- 마케터 및 대행사: 다계정 다채널을 효율적으로 운영해야 하는 실무자.
- 마케팅 초심자: 무엇을 써야 할지 모르는 신규 운영자.

## 9. 주의 표현 가이드
- "최고", "유일", "보장", "1위 확정" 등 절대적/과장 표현은 표시광고법 위험이 있으므로 피한다.
- 수치와 후기는 검증된 실제 데이터와 동의받은 사례로만 사용한다.
